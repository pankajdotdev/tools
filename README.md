
This repository contains a collection of small yet powerful utility tools that help automate common tasks and improve productivity.

Whether you're working on file processing, scripting, or automation — these tools are designed to save time and reduce repetitive work.

---

## 📂 Project Structure

```
tools-repo/
│
├── file-tools/        # PDF merge, split, rename, etc.
├── automation/        # Scripts for automation tasks
├── system-utils/      # OS related utilities
├── scripts/           # General purpose scripts
└── README.md
```

## ⚙️ Usage

1. Clone the repository:

```bash
https://github.com/pankajdotdev/tools.git
```

2. Navigate to the tool you want to use:

```bash
cd tools```

3. Run the script:

```bash
python script_name.py - according to you need
```
